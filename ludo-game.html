LUDO GAME

The Ludo Game Project is a web-based version of the classic board game, where players can enjoy a colorful and interactive experience. Built using HTML, CSS, and JavaScript, the project allows 2 to 4 players to roll dice and move their pieces around the board with the goal of reaching the center first. The game features vibrant colors, smooth animations, and intuitive controls, offering a seamless and engaging experience for players of all ages. It also includes player turn indicators and real-time movement of pieces, enhancing the gameplay.
