TEMPERATURE CONVERTER

This Temperature Converter App allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin with a sleek, colorful interface. It features a Light/Dark Mode toggle, real-time date and time display, and saves the theme preference using localStorage**. The app provides a smooth, interactive user experience for temperature conversions and mode switching.

The app is built using HTML, CSS, and JavaScript for structure, styling, and functionality, respectively.
